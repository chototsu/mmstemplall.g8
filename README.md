mmstemplall.g8
==============

MikuMikuStudio template for all targets.

## First
Install giter8
https://github.com/n8han/giter8

## Usage
Create a project:

    g8 chototsu/mmstemplall

Run on PC(Windows, MacOS X, Linux)

    cd (project folder)
    sbt
    run

Run on Android

    cd (project folder)
    sbt
    project android-project
    run

Create IntelliJ IDEA Project

    cd (project folder)
    sbt
    gen-idea
